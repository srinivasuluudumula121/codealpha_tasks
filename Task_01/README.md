# Task 01 – Local Service Business Website Redesign

## Project Name

Partha Dental Clinic – Website Redesign

---

## Project Overview

This project focuses on redesigning a local dental clinic website with a strong emphasis on trust building, clarity, and appointment conversion.

The goal of the redesign was to improve website usability, strengthen visual hierarchy, and create a more professional experience for first-time visitors.

The redesigned website maintains the clinic’s brand identity while improving navigation, responsiveness, and lead generation.

---

## Problem Statement

Many local healthcare websites struggle with:

* unclear messaging
* weak calls-to-action
* poor visual hierarchy
* information overload
* poor mobile responsiveness

As a result, users often leave the website without contacting the clinic or booking appointments.

This redesign addresses those issues through better UX structure and conversion-focused design.

---

## Project Goal

The website redesign focuses on:

* Clear service communication
* Better trust building
* Improved appointment conversion
* Responsive desktop and mobile experience
* Better user navigation and readability

---

## Target Users

### Families

Parents looking for trusted dental care for family members.

### Working Professionals

Users needing fast consultation and appointment access.

### First-Time Visitors

Patients unfamiliar with the clinic who require reassurance before booking.
