# Task 02 – Mobile Appointment Booking App

## Project Name

Bhai Looks Saloon – Mobile Appointment Booking App

---

## Project Overview

This project focuses on designing a mobile-first appointment booking experience for a local salon business.

The goal was to simplify the appointment booking process, reduce booking friction, and improve the overall customer experience through an intuitive and easy-to-use mobile interface.

The application was designed to help users quickly browse services, select barbers, choose appointment slots, and confirm bookings efficiently.

---

## Problem Statement

Many local businesses still manage appointments using:

* phone calls
* WhatsApp messages
* handwritten registers

This creates several problems:

* missed bookings
* scheduling confusion
* poor customer experience
* long waiting times
* operational inefficiencies

This mobile booking system solves these challenges through a structured and user-friendly digital appointment experience.

---

## Project Goal

The appointment booking app focuses on:

* Fast booking experience
* Reduced user friction
* Better scheduling clarity
* Mobile-first usability
* Improved customer convenience
* Clear appointment flow

---

## Target Users

### Returning Customers

Users who already know their preferred barber and want quick appointment booking.

### First-Time Customers

Users unfamiliar with the salon staff who require recommendations and booking guidance.
